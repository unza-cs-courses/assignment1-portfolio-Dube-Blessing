About Me

I am Blessing Dube, a Software Engineering student with a strong passion for building innovative digital solutions that solve real-world problems. My journey into technology started with curiosity about how websites and applications work, which gradually developed into a deep interest in software development, system design, and web technologies. I enjoy transforming ideas into functional systems through code, whether it’s building web applications, designing user-friendly interfaces, or developing scalable digital platforms. Alongside my academic journey, I am the founder of BrookBurst Marketing, a digital marketing company focused on helping businesses grow their online presence through strategic design, branding, and marketing solutions. Through this venture, I have gained practical experience in graphic design, digital branding, and understanding how technology and marketing work together to drive business success. My interests include web development, software architecture, and building platforms that improve access to education and digital services. I am constantly learning new technologies and improving my skills to become a well-rounded software engineer. My goal is to build impactful software products and contribute to innovative solutions that empower people, businesses, and communities in the digital world.

Point of Sale System

I developed a Point of Sale (POS) system to help businesses manage sales transactions more efficiently. The system allows users to add products, calculate totals, and generate receipts during checkout. It was built using Python for the backend functionality, with HTML and CSS used to create the user interface. The system helps reduce manual errors and speeds up the sales process for small businesses.

Library Management System

I developed a Library Management System to help manage books, students, and borrowing activities efficiently. The system allows users to add books, issue and return them, and keep track of available and borrowed items. It was built using Python for the backend logic and HTML and CSS for the user interface. The system helps libraries organize their records and reduce manual work in managing books and transactions.
